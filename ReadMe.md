# git学习

## git 安装()
$ sudo apt-get install git

## 创建git仓库 初始化
$ mkdir [name]

$ cd [name]

$ git init
  
## Git clone 拷贝一个 Git 仓库到本地，让自己能够查看该项目，或者进行修改。
git clone [url]

## 基本快照 （Git 的工作就是创建和保存你的项目的快照及与之后的快照进行对比）
 ###git add

 git add 命令可将该文件添加到缓存，如我们添加以下文件：

  $ touch README

  $ touch hello.php
